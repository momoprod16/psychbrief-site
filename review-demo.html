TikTok app-review walkthrough

End-to-end TikTok integration flow.

This page illustrates the Psych Brief flow: the public site (this domain), Login Kit authorization for user.info.basic, and the OAuth callback. It also previews a planned Content Posting API draft upload (video.upload, inbox draft) that will be submitted separately. The workflow uses only authorized creator access and creator-owned MP4 drafts.

Website URL shown for review: this domain/

No Direct Post. Psych Brief uploads to the creator’s TikTok inbox as a draft only. The creator reviews and publishes inside TikTok. Direct Post / video.publish are not used unless that capability is separately requested and approved.

The creator connects the Psych Brief TikTok account. Token values are stored server-side/local only and are never displayed in the interface.
